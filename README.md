# 🎯 AI Resume Matcher PRO

A premium, glassmorphic Streamlit application that uses AI to match resumes with job descriptions.

## ✨ Features
- **Modern UI**: Sleek dark theme with glassmorphic elements.
- **Interactive Metrics**: Custom circular gauge for match percentage.
- **PDF Extraction**: Robust text extraction from PDF resumes.
- **AI Insights**: (Placeholder) Strategic advice and skill gap analysis.

## 🚀 How to Run Locally

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Run the App**:
   ```bash
   streamlit run app.py
   ```

## 🌐 Deployment
This app can be easily deployed to **Streamlit Community Cloud**:
1. Push this repository to GitHub.
2. Sign in to [Streamlit Cloud](https://share.streamlit.io/).
3. Connect your GitHub repo and deploy!
